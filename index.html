purple-events

A libpurple events handling plugin and library

purple-events allows a fine-grained control over libpurple events (buddy signing on, new message).
It does not dispatch any event to any system, a dispatching plugin is needed.

Dependencies

  • Pidgin (2.6.0 or later)

Releases download

Visit the downloads page to find releases (tar.xz) and automatic GitHub tag tarball/zip.
